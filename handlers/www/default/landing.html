About Docker Playclouds

This is a simplified version of Play with Docker (PWD), originally created by Marcos Liljedhal and Jonathan Leibiusky and sponsored by Docker Inc.

Our streamlined version provides a one session = one container experience, making it easier to get started with Docker without the complexity of managing multiple instances. It uses Docker-in-Docker (DinD) technology to provide a full containerized environment directly in your browser.
